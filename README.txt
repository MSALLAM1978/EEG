# EEG Analysis

This Python script provides methods to analyze EEG data, specifically for channel selection, coherence calculation, and phase lag index (PLI) computation.

## Overview

This script performs:
- **Channel Selection**: Keeps only relevant EEG channels.
- **Coherence Calculation**: Computes coherence between EEG signals.
- **Phase Lag Index (PLI) Calculation**: Measures phase synchronization.

## Requirements

- Python 3.x
- NumPy
- SciPy

Install dependencies with:

```bash
pip install numpy scipy
